    Claude outage FAQ

    Is Claude down right now?

    This page checks Anthropic's official status feed live when it loads, and again every 30 seconds. If the big answer reads NO, Claude is reporting problems; YES means all systems are operational.

    Is it Claude that's down, or my internet?

    Quick checks: load another website to rule out your connection, try Claude in an incognito window or on a different network, and clear your cache. If this page shows Claude as up but it still fails for you, the problem is likely local — your network, a browser extension, or your account.

    What's the difference between "Claude" and "Claude Code" being down?

    Claude (claude.ai) is the chat app and the API; Claude Code is the command-line / agent tool for developers. They can fail independently — the API can be degraded while the website is fine, or vice versa. The service breakdown above shows each component separately.

    Where does this site get its status data?

    Straight from Anthropic's official status page at status.claude.com — the same source Anthropic publishes. We don't run our own probes or guess, so what you see here matches the official report.

    Will you email me when Claude is back up?

    Yes — that's the whole point. Pop your email in the signup box above and we'll send a single message the moment Claude recovers, so you can get straight back to work.

    How often does Claude go down?

    Like any large online service, Claude is up the vast majority of the time but has the occasional incident — usually brief, and sometimes affecting only one component (a single model, or just the API). For the official history of past incidents, see status.claude.com.
